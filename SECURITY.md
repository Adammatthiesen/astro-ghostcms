# Security Policy

## Supported Versions

| Version | Supported          |

## Reporting a Vulnerability

If you find an issue with our security please report it to [issues@astro-ghostcmx.xyz](mailto:issues@astro-ghostcms.xyz) or Open a Issue on GitHub
